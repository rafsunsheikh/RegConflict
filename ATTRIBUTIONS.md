# Attributions

When using the TRIAG dataset, please preserve the following per-source attribution requirements. This file lists every source organisation whose content (or metadata) is included in this release, along with the attribution string they require.

For sources where the document content is redistributed, attribution applies to derived works (e.g., publications, downstream datasets) that reuse this material. For sources where only metadata is included, attribution acknowledges the original document origin.

## How to use this file

If your paper or downstream dataset uses material from a specific source listed below, include that source's attribution string. A consolidated attribution that covers all sources used is acceptable; per-document attribution is not required.

A minimal acknowledgement format:

> This dataset includes regulatory content from {list of sources used}. See ATTRIBUTIONS.md in the dataset release for required attributions.
